ClawTVOS — GitHub Pages 与域名（本机执行）

1) 登录 GitHub CLI
   gh auth login -h github.com

2) 创建远程仓库、推送 main、启用 Pages（分支 main、根目录）
   cd /path/to/clawtvos.site
   bash scripts/bootstrap-github-pages.sh
   成功后用浏览器打开脚本输出的 https://<用户名>.github.io/clawtvos.site/ 做冒烟测试。

3) 在 GitHub 绑定自定义域名（建议在配置 DNS 之前先在 GitHub 保存域名，降低被抢注风险）
   任选其一：
   - 网页：仓库 Settings → Pages → Custom domain 填 clawtvos.com → Save
   - 或：bash scripts/set-custom-domain.sh
   仓库已包含 CNAME 文件，内容与 clawtvos.com 一致。

4) 在 Spaceship 按 spaceship-dns-records.txt 添加记录（将 YOUR_GITHUB_USERNAME 换成你的用户名）。

5) 等待 DNS 生效后，在 Settings → Pages 勾选 Enforce HTTPS（若灰色请隔一段时间再试）。
